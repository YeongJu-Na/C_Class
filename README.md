* 변수

* 상수
- 리터럴 상수 -> int a=3+4;	→ 메모리 공간차지함 → | 3 | 4 | 7(a) |

* 연산자: shift연산자
- 비트shift로 <<1은 *2, >>1은 /2	 ⇒ c++에서는 입출력 관련 문자

* 입력받기: scanf
- 서식문자열?에는 다른내용(prompt ?) 들어가면 안됨
- 변수의 타입과 다른 값 입력 시 오류 → 이러한 오류발생 가능성이 많아서 윈도우즈 프로그램에선 다른 것으로 대체하여 사용 → 추후 학습할 것
